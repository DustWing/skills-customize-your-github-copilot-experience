# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master text processing techniques by working with strings, file I/O operations, and text manipulation. You'll learn how to read, process, and transform text data using Python's powerful string methods and file handling capabilities.

## 📝 Tasks

### 🛠️ String Manipulation and Analysis

#### Description
Write a program that reads a text file and performs various string operations to analyze and transform the text content. You'll practice accessing string methods and processing text data.

#### Requirements
Completed program should:

- Read text from a file
- Perform string operations (convert case, search, replace, split)
- Count occurrences of specific words or characters
- Display the processed results clearly

### 🛠️ File I/O and Text Transformation

#### Description
Create a program that reads one or more text files, applies transformations to the content, and writes the results to a new file. This reinforces file handling best practices and text manipulation skills.

#### Requirements
Completed program should:

- Open and read from input file(s)
- Apply text transformations (filtering, formatting, sorting)
- Write processed output to a new file
- Handle edge cases gracefully
- Include informative messages about the processing steps
